# Engineering calculator

[English version](README.md)

Десктопное приложение для вычисления математических выражений и построения графиков функций на **C++ и Qt Widgets**. Лексер и рекурсивный парсер написаны вручную — сторонние математические библиотеки не используются.

## Возможности

- Вычисление арифметических выражений с приоритетом операторов
- Научные функции: `sin`, `cos`, `tan`, `log`, `ln`, `sqrt`, `!`
- При обнаружении переменной `x` автоматически строится график функции
- История вычисленных выражений доступна через меню

---

## Лексер и парсер

Обработка входной строки проходит в два этапа.

**Лексер** обходит строку и преобразует её в плоскую последовательность типизированных токенов — числа, операторы, имена функций, скобки, переменная `x`.

**Парсер** потребляет поток токенов методом рекурсивного спуска, в котором каждое правило грамматики реализовано отдельной функцией:

```
expression  →  term  (('+' | '-')  term)*
term        →  factor  (('*' | '/')  factor)*
factor      →  number | variable | '(' expression ')' | func '(' expression ')' | '-' factor | factor '!'
```

## Стек

- **C++17**
- **Qt 6**
- **QCustomPlot** — отрисовка графиков
- **qmake**

Паттерн MVC: `MainWindow` — представление, `Controller` — маршрутизация выражений, `Core` + `ParserMath` — модель.

## Сборка

```bash
git clone https://github.com/Pavel-Sedlovec/engineering-calculator.git
cd engineering-calculator
qmake calculator.pro
make -j$(nproc)
./calculator
```
